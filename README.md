# Organización del repositorio

La rama `main` se utiliza únicamente como punto principal de documentación y organización del laboratorio.

Cada proyecto se desarrolla de manera independiente en ramas separadas para mantener una estructura limpia y ordenada.

---

# Objetivos del laboratorio

- Implementar componentes reutilizables
- Utilizar props para comunicación entre componentes
- Manejar estados con `useState`
- Implementar eventos en React
- Mantener una estructura limpia del proyecto
- Aplicar buenas prácticas básicas de desarrollo frontend
